# Privacy Statement

`library` is a local-first library designed to run completely on your local machine.

## No Telemetry or Data Collection

- **Zero Data Collection**: `library` does not collect, log, transmit, or upload any personal data, usage metrics, device information, or telemetry.
- **Local Execution**: All system information queries, process scans, and hardware configurations (such as monitors, battery, disk, and OpenRGB integrations) are executed entirely on your local machine.
- **Offline by Design**: The library does not connect to any external web servers, APIs, or remote services at runtime.

## Third-Party Integrations

If you are using `library` alongside external integrations (such as OpenRGB server configurations), data transmissions are limited to communication between your local applications and your local hardware control daemons. We do not control or access this data.
