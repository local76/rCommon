# Support Guide

Welcome! If you need help with `library` or have questions about integrating it into your `apps` consumer application, please review the support channels below.

## 🔍 Self-Service Help

Before opening an issue or asking for support, please check the following resources:

1. **Architecture & Guidelines**: Review [ARCHITECTURE.md](file:///C:/Users/jeryd/Synology/Home/Projects/local76/library/ARCHITECTURE.md) to understand the taxonomy and design.
2. **Usage Examples**: Check the `🚀 Usage` section in [README.md](file:///C:/Users/jeryd/Synology/Home/Projects/local76/library/README.md).
3. **Rust Documentation**: Generate and read the local library docs using Cargo:
   ```bash
   cargo doc --open --all-features
   ```

## 💬 Community & Issue Tracker

For bugs, feature requests, or usage questions:

- **Bugs & Features**: Search existing reports or file a new issue on the [GitHub Issues tracker](https://github.com/local76/library/issues).
- **Discussions**: For general design discussions or suggestions, use the repository's Discussion forum.

Please provide a clear description, reproduction steps, and platform context (Windows/Linux) when reporting issues.
